# NextFlix 🎬

NextFlix is a Movie Recommendation System built using Machine Learning.

Features:
- Content-based filtering (TF-IDF)
- Collaborative filtering
- Movie posters using TMDB API
- Trending movies
- Movie trailers
- Netflix-style UI
- Streamlit web application

Tech Stack:
- Python
- Pandas
- Scikit-Learn
- Streamlit
- TMDB API

Run the app:

streamlit run app.py
